Eleventy หรือ 11ty คือ “static site generator” แบบโอเพนซอร์สที่เขียนด้วย JavaScript ใช้สำหรับสร้างเว็บไซต์แบบไฟล์คงที่ (HTML, CSS, JS) จากเทมเพลตและไฟล์คอนเทนต์ เช่น Markdown หรือ Nunjucks โดยทำงานบน Node.js

## Eleventy คืออะไร

Eleventy (ย่อว่า 11ty) เป็นซอฟต์แวร์สำหรับ “คอมไพล์” ไฟล์เทมเพลตและคอนเทนต์ของคุณออกมาเป็นเว็บไซต์ static คล้าย Hugo, Jekyll, Astro แต่โฟกัสที่ความเรียบง่ายและยืดหยุ่นสูง ไม่ผูกกับเฟรมเวิร์ก front‑end ใดเป็นพิเศษ. ตัวมันทำงานบนรันไทม์ JavaScript (ปกติคือ Node.js เวอร์ชัน 18 ขึ้นไป) และเป็นโอเพนซอร์ส คุณสามารถติดตั้งผ่าน npm แล้วเรียกใช้คำสั่ง `eleventy` เพื่อ generate ไฟล์ไซต์ออกมาได้

## หลักการทำงานโดยย่อ

แนวคิดหลักคือคุณเก็บคอนเทนต์ในรูปแบบไฟล์ เช่น Markdown หรือ HTML เทมเพลต แล้ว Eleventy จะอ่านไฟล์เหล่านี้ นำไปผ่าน template engine (เช่น Nunjucks, Liquid ฯลฯ) แล้วสร้างโฟลเดอร์เอาต์พุตที่เต็มไปด้วยไฟล์ HTML static พร้อมใช้งาน. ตัวอย่างการเริ่มต้นที่ผู้ใช้ชอบอ้างถึงคือเพียงติดตั้ง Eleventy, สร้างไฟล์ Markdown แค่ไฟล์เดียว แล้วสั่ง `eleventy` ก็จะได้เว็บไซต์อย่างง่ายทันที ซึ่งเป็นเหตุผลหนึ่งที่ถูกมองว่า “เบาและง่าย” กว่า static generator ที่อิง React บางตัว

การใช้งาน Eleventy ที่พบได้บ่อยคือทำบล็อกส่วนตัว เว็บไซต์เอกสารประกอบ (documentation) และเว็บไซต์องค์กรที่ไม่ต้องการ backend ซับซ้อน เพราะไฟล์ที่ได้เป็น HTML static ทำให้โฮสต์บนบริการอย่าง Netlify หรือแพลตฟอร์ม static hosting อื่น ๆ ได้ง่ายและโหลดเร็ว. ในงานรีแฟกเตอร์เว็บเก่า นักพัฒนาบางคนก็ใช้วิธีดึงเว็บเก่าออกมาเป็น HTML แล้วค่อย ๆ ย้ายโครงสร้างเข้ามาใน Eleventy บนโฮสติ้งใหม่ เช่น Netlify เพื่อให้จัดการได้ง่ายขึ้นและใช้เทคโนโลยีสมัยใหม่มากขึ้น

Eleventy ได้รับความนิยมเพราะโครงสร้างโปรเจกต์เรียบง่าย ไม่บังคับ opinion มากนัก คุณสามารถใช้หลายภาษาเทมเพลตในโปรเจกต์เดียว และสามารถใช้ HTML ธรรมดาได้โดยไม่ต้องเขียน JavaScript ครอบเหมือนบางเฟรมเวิร์กที่ใช้ React เป็นฐาน. อีกทั้งมีความเร็วในการ generate ไซต์ที่ดี และเรียนรู้ไม่ยากสำหรับผู้ที่คุ้นกับ HTML/Markdown อยู่แล้ว ทำให้เดเวลอปเปอร์หลายคนเลือกใช้ในโปรเจกต์ที่ต้องการความเร็วและความเบา มากกว่าความเป็น SPA เต็มรูปแบบ
